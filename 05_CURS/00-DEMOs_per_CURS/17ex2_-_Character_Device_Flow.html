Curs 17 • Comunicare User Space ↔ Kernel Space

Character Device = File Special

Aplicațiile din user space comunică cu driver-ele kernel folosind /dev/ entries. Fișierele device sunt "porți" către funcțiile driver-ului: open(), read(), write(), close().

User ↔ Kernel Data Transfer

copy_to_user()

Kernel → User Space

Folosită în read() pentru a trimite date din kernel buffer către aplicație.

copy_to_user(user_buf, kernel_buf, count);
copy_from_user()

User Space → Kernel

Folosită în write() pentru a primi date de la aplicație în kernel buffer.

copy_from_user(kernel_buf, user_buf, count);

IMPORTANT: Nu poți accesa direct memoria user space din kernel! Folosește întotdeauna copy_to_user() / copy_from_user() pentru a evita kernel panic și security vulnerabilities.
