# M07: Security Audit Framework

> **Nivel:** MEDIUM | **Timp estimat:** 25-35 ore | **Bonus K8s:** +10%

## Descriere
Framework pentru audit de securitate: verificare permisiuni, configurări nesigure, useri, SSH, firewall și compliance policies.

## Cerințe
1. **User audit** - conturi, permisiuni, parole vechi
2. **File permissions** - world-writable, SUID/SGID
3. **SSH config** - best practices check
4. **Firewall** - reguli și porturi deschise
5. **Services** - servicii inutile running
6. **Compliance** - check-uri CIS Benchmark
7. **Raport** - cu severitate și remedieri
