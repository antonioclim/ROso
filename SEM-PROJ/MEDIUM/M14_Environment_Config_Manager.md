# M14: Environment Config Manager

> **Nivel:** MEDIUM | **Timp estimat:** 25-35 ore | **Bonus K8s:** +10%

## Descriere
Manager pentru configurări de mediu: environment variables, dotfiles, profiles multiple și sincronizare între mașini.

## Cerințe
1. **Profile management** - dev, staging, prod
2. **Dotfiles** - versionare și sync
3. **Secrets** - management sigur (encrypted)
4. **Switch** - schimbare rapidă între profile
5. **Diff** - comparație între environments
6. **Export/Import** - portabilitate
7. **Templates** - generare din template-uri

*Proiect MEDIUM | Sisteme de Operare | ASE-CSIE*
