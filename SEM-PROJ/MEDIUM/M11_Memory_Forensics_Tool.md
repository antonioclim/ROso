# M11: Memory Forensics Tool

> **Nivel:** MEDIUM | **Timp estimat:** 25-35 ore | **Bonus K8s:** +10%

## Descriere
Tool pentru analiza utilizării memoriei: memory maps, shared memory, memory leaks detection și profiling.

## Cerințe
1. **Memory maps** - vizualizare /proc/[pid]/maps
2. **RSS/VSZ tracking** - evoluție în timp
3. **Shared memory** - segmente și utilizatori
4. **Leak detection** - identificare creștere continuă
5. **Cache analysis** - page cache, buffers
6. **Per-process breakdown** - top consumers detaliat
7. **Raport** - recomandări optimizare
