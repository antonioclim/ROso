# M04: Network Security Scanner

> **Nivel:** MEDIUM | **Timp estimat:** 25-35 ore | **Bonus K8s:** +10%

## Descriere
Scanner de securitate pentru rețea: descoperire hosturi, port scanning, verificare vulnerabilități comune și raport de securitate.

## Cerințe Obligatorii
1. **Host discovery** - ping sweep, ARP scan
2. **Port scanning** - TCP/UDP, range configurabil
3. **Service detection** - identificare servicii pe porturi
4. **Vulnerabilități comune** - verificare porturi nesigure, banner grabbing
5. **Raport** - text, HTML, JSON cu findings
6. **Comparație** - diff între scanări
7. **Scheduling** - scanări periodice

## Opționale
8. **SSL/TLS check** - verificare certificate
9. **DNS enumeration** - subdomain discovery
10. **Compliance check** - verificare politici
