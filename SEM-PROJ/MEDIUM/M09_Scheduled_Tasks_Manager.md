# M09: Scheduled Tasks Manager

> **Nivel:** MEDIUM | **Timp estimat:** 25-35 ore | **Bonus K8s:** +10%

## Descriere
Manager centralizat pentru task-uri programate: wrapper peste cron cu logging, alerting, dependencies și UI text.

## Cerințe
1. **Task management** - add, remove, enable, disable
2. **Scheduling** - sintaxă cron + intervale simple
3. **Dependencies** - task A după task B
4. **Logging** - output și durata fiecărui run
5. **Alerting** - la eșec sau timeout
6. **Lock/mutex** - prevenire execuție paralelă
7. **Dashboard** - status și istoric task-uri
