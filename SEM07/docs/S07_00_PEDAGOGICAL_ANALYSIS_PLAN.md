## Integrarea obiectivelor de învățare

Această sesiune de evaluare verifică TOATE obiectivele de învățare din SEM01–SEM06:

### SEM06: Integrare de tip capstone
- LO6.1–LO6.8: Integrare de sistem, deployment, monitorizare

## Filosofia evaluării

1. **Autenticitate** — toate evaluările testează competențe aplicabile în situații reale
2. **Echitate** — rubrici clare, aplicare consecventă
3. **Anti‑plagiat** — verificare orală, randomizare, detecție AI
4. **Feedback** — feedback detaliat pentru îmbunătățirea învățării

## Distribuția pe taxonomia Bloom (cumulativ)

| Nivel | Țintă | Metodă de evaluare |
|-------|--------|--------------------|
| Reamintire | 10–15% | Alegere multiplă, definiții |
| Înțelegere | 20–25% | Întrebări de explicare, comparație |
| Aplicare | 25–30% | Exerciții practice, codare |
| Analiză | 15–20% | Depanare, analiză de loguri |
| Evaluare | 5–10% | Review de cod, critică de design |
| Creare | 5–10% | Proiect, soluții originale |
