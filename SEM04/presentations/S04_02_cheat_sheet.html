üîç GREP - Options

‚ö†Ô∏è -c counts LINES, not occurrences! Use grep -o | wc -l
üîç GREP - Examples
# Search case-insensitive
grep -i 'error' log.txt

# Everything that is NOT a comment
grep -v '^#' config.txt

# Extract IP addresses
grep -oE '([0-9]{1,3}\.){3}[0-9]{1,3}' log.txt

# Recursive in code
grep -rn 'TODO' --include='*.py' .
‚úÇÔ∏è SED - Substitution
SyntaxEffect
s/old/new/First occurrence
s/old/new/gAll on line
s/old/new/iCase insensitive
-iEdit in-place (no backup!)
-i.bakEdit with backup
/pattern/dDelete lines
&Entire match

üí° Alternative delimiters: s|/usr|/opt| or s#old#new#
‚úÇÔ∏è SED - Examples
# Replace all occurrences
sed 's/localhost/127.0.0.1/g' config.txt

# Delete comments
sed '/^#/d' config.txt

# Delete blank lines
sed '/^$/d' file.txt

# Edit in-place with backup
sed -i.bak 's/old/new/g' file.txt
‚ö†Ô∏è sed ... > file EMPTIES the file! Use -i.bak
üìä AWK - Variables
VariableContent
$0Entire line
$1, $2...Fields
$NFLast field
NFNumber of fields
NRLine number (global)
FNRLine number (per file)
FSInput Field Separator
OFSOutput Field Separator
‚ö†Ô∏è print $1 $2 = concatenated | print $1, $2 = with space!
üìä AWK - Examples
# Print column 2
awk '{print $2}' file.txt

# CSV - column 3
awk -F',' '{print $3}' file.csv

# Skip header
awk 'NR > 1 {print}' file.csv

# Sum column
awk '{sum += $1} END {print sum}' file

# Count per category
awk '{c[$1]++} END {for(k in c) print k, c[k]}'

üîó Combined Pipelines
