## 1. Resurse Interactive de Testare

### regex101.com
Cel mai util instrument pentru învățarea și testarea expresiilor regulate.

**Caracteristici:**
- Vizualizare în timp real a match-urilor
- Explicații detaliate pentru fiecare parte din pattern
- Suport pentru multiple dialecte (PCRE, Python, JavaScript, Go)
- Salvare și partajare pattern-uri
- Quick reference integrat

**Recomandare:** Testează ORICE regex aici înainte de a-l folosi în scripturi.

**Link:** https://regex101.com

### explainshell.com
Explică comenzile shell detaliat, argument cu argument.

**Utilizare:** Lipește o comandă complexă și primești explicație pentru fiecare parte.

**Link:** https://explainshell.com

### ShellCheck Online
Verificator de scripturi Bash online.

**Link:** https://www.shellcheck.net

## 9. Instrumente Locale

### man pages
Nu uita de documentația locală:
```bash
man grep
man sed
man awk
man bash
```

### info pages
Documentație mai detaliată pentru unelte GNU:
```bash
info grep
info sed
info gawk
```

### --help
Ajutor rapid pentru orice comandă:
```bash
grep --help
sed --help
awk --help
```
